# Shellty Lingo — ข้อมูลหน้าร้าน (TH)

**ชื่อ:** Shellty Lingo

**คำโปรย:** เรียน สนทนา และพัฒนาได้ทุกวัน

**คำอธิบายสั้น:** เรียนภาษาอังกฤษหรือภาษาไทยด้วยบทเรียนสั้น การทบทวนอัจฉริยะ และบทสนทนาแบบข้อความกับ AI ที่ปลอดภัย

**คำอธิบาย:** เรียนตามแผนประจำวัน ฝึกทักษะตามระดับ และทบทวนคำศัพท์ในเวลาที่เหมาะสม ฝึกสถานการณ์จริงผ่านบทสนทนาแบบข้อความ เรียนรู้อักษรไทย และติดตามความก้าวหน้าของแต่ละหลักสูตรแยกกัน Shellty Lingo แสดงขีดจำกัดอย่างชัดเจน ให้รายงานคำตอบของ AI และควบคุมการแจ้งเตือนกับข้อมูลส่วนตัวได้

**คำสำคัญ:** ภาษาอังกฤษ, ภาษาไทย, เรียนภาษา, คำศัพท์, ทบทวน, สนทนา

**หมายเหตุเบต้า:** ฟีเจอร์ทดลองอาจเปิดให้ใช้เป็นขั้นตอนและปิดจากระยะไกลได้ เสียงที่บันทึกเพื่อฝึกพูดจะไม่ออกจากอุปกรณ์
