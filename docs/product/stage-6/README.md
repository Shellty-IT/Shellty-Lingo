# Etap 6 — język tajski

Dodano osobny, recenzowany model znaków tajskich (`ThaiScriptUnit`) obejmujący spółgłoski, samogłoski, sylaby, cyfry i reguły tonów. Mobilna ścieżka prezentuje zapis tajski, nazwę, transliterację, klasę/ton, znaczenie, przykład i systemowy odsłuch. Użytkownik może wyłączyć transliterację; ustawienie jest zapisywane osobno dla kursu TH.

Publikowane są wyłącznie jednostki oznaczone jako sprawdzone przez eksperta. Interfejs jasno informuje, że MVP ćwiczy rozpoznawanie tonów, ale nie obiecuje automatycznej oceny wymowy.

Demonstracja: wybierz kurs tajski → Nauka → Alfabet i tony → przełącz transliterację → odsłuchaj znak.
